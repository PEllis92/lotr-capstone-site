Lord of the Rings Fan Website: Tales of Middle-Earth

This is a Lord of the Rings themed fan website created as part of my Capstone Web Development project for the school UWG.

The site provides information about Middle-Earth locations, features LOTR-themed media, and includes a form for fans to join a fellowship group.

Features:
- Responsive HTML & CSS website
- Multiple pages (Home, Landmarks, Features, Join Form, etc.)
- Images and LOTR-themed design
- Audio/Trailer embeds
- Form submission page
- Accessibility features (alt text, ARIA labels)
- W3C HTML & CSS validation completed

Purpose:
- Website planning and design
- HTML and CSS styling
- Image and media integration
- Accessibility & usability testing
- Deployment to GitHub Pages

How to View the Site:
The website is deployed using GitHub Pages.

Navigate:
- Browse pages from the navigation menu
- View Middle-Earth locations and media
- Experience LOTR-themed content and audio
- Fill out the form to “Join the Fellowship”

Tech Stack:
- HTML5 & CSS3
- Visual Studio Code
- GitHub & GitHub Pages
- W3C Validators (HTML & CSS)
